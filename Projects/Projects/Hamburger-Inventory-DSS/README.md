# Hamburger Branch Inventory Decision Support System (DSS)

This project is a Decision Support System (DSS) developed to help a hamburger restaurant branch maintain product continuity by determining optimal replenishment quantities during shipment ordering.

The system analyzes sales data and inventory levels to calculate the required order quantities, ensuring that stock shortages are minimized while avoiding overstock situations.

## Objectives
- Maintain product availability
- Prevent stock-outs
- Optimize shipment order quantities
- Support data-driven inventory decisions

## Features
- Sales-based demand estimation
- Inventory tracking
- Automated order quantity calculation
- Decision support interface

## Technologies Used
- Microsoft Excel
- VBA (Visual Basic for Applications)
- Data Analysis Techniques

## Project Type
Group Project
